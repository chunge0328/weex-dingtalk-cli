## weex-dingtalk-cli

> A simple CLI for scaffolding Dingtalk Weex Microapp projects.

## Install

```bash
$ npm install -g weex-dingtalk-cli
```

## Usage

```bash
$ dingtalk
```

```bash
$ dingtalk init
```

```bash
$ dingtalk init simple icepy
```

```bash
$ dingtalk init simple icepy --offline
```

```bash
$ dingtalk list
```

```bash
$ dingtalk search
```
